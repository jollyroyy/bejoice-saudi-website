# BeJoice Saudi - Premium Freight Forwarding Website

🌍 **Delivering Dreams, Building Trust**

A premium, high-performance website for BeJoice Shipping LLC — a fast-growing International Freight Forwarding service provider.

## Features

- 🚢 **Parallax Scrolling** — Immersive 8K background imagery for Sea, Air, and Land freight
- ✨ **Glassmorphism Design** — Modern frosted-glass UI panels
- 📊 **Animated Counters** — Real-time counting stats (5000+ deliveries, 80+ countries)
- 🎯 **Auto-hiding Navbar** — Seamless scrolling experience
- 📱 **Fully Responsive** — Mobile-first design with premium aesthetics
- 🏢 **Client Testimonials** — Dual-direction animated marquee showcasing global partners
- 📧 **Lead Generation Form** — High-contrast premium contact form
- 🔍 **SEO Optimized** — Meta tags, semantic HTML, and descriptive title

## Tech Stack

- **Vite** — Lightning-fast build tool
- **Vanilla JS** — Zero-dependency frontend
- **CSS3** — Custom animations, parallax, glassmorphism
- **Google Fonts** — Cinzel + Montserrat typography

## Getting Started

```bash
npm install
npm run dev
```
